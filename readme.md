1. Créer votre environement avec requirements.txt
    1.1. Assurer vous que c'est python 3.8
2. Partir le server avec server.py 
    2.1. verrifer votre root, c'est comme tp4 
3. Pour ajouter les charts:
    3.1. Créer un file.py dans charts.py pour ajouter votre code
    3.2. vous pouvez utiliser utils pour les preprocess ou d'autre chose
    3.3. visiter Sidebar dans components et app.py et lire les commandaires pour adapter les layouts
4. vérifier la structure de votre file avec role_heatmap si ca ne fonctionne pas
